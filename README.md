## ⬇️ How to get New Emojis:

- Quit Ripcord completely.
- Replace `twemoji.ripdb` file in the Ripcord folder with [this](https://github.com/cool-dev-code/ripcord-emoji-fix/releases) file.
- Now Open **Ripcord**.
- You are Done. Enjoy :)
